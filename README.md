# 起動方法
1. フロントエンドのinstall (./fontend/ で以下を実行)
```
yarn install
```

2. バックエンドのinstall (./backend/ で以下を実行)
```
yarn install
```

3. アプリケーションを起動 (./ で以下を実行)
```
yarn install
yarn start
```
